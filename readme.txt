== Description ==

iCompare is designed specifically for affiliate marketers to easily add products from eBay and Amazon into their post. The products will be displayed in a table comparing prices from eBay and Amazon. That means cashing on the eBay and Amazon commission you deserved, without spending tons of time creating affiliate links is easier than ever.

Free version will have the following features:
1. Two e commerce sites: Ebay and Amazon
2. Links are long and not traceable

== Installation ==

Before installing the plugin, please register first for a developer account on eBay and affiliation in Amazon. [Complete guide for installation](http://icompareplugin.com/how-to-get-affiliate-id/ "Complete guide for installation")
